# Asset Checklist

The current project uses procedural placeholders so the game can run before final art is ready. For the next pass, send these assets as transparent PNG or sprite sheets:

- Empty 6x6 fantasy board background
- Populated 6x6 board background
- Blue castle, red castle
- Forest, mountain, river, gold mine, crystal mine, ruins, shrine, portal
- Four hero portraits: Wizard, Knight, Dark Lord, Ranger
- Blue and red avatars or faction portraits
- UI frame pieces for gold borders and glowing buttons
- Tile capture, attack, mana gain, and castle hit effects

Recommended mobile source size: at least 2048 px tall for full-screen backgrounds, and 256-512 px for individual tile props. Keep effects separate from the board so both Skia and PixiJS can animate them.
